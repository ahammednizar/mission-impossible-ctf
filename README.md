# Project Mission Impossible – Boot2Root CTF

A **story-driven Boot2Root Capture The Flag (CTF)** challenge inspired by the *Mission Impossible* franchise.

This vulnerable lab provides a realistic penetration testing environment where players must perform reconnaissance, exploit web vulnerabilities, pivot between users, and escalate privileges to root.

---

## Challenge Overview

The challenge contains a vulnerable Debian virtual machine with intentionally misconfigured services and web applications.

Participants must chain vulnerabilities together to obtain **root access and capture the final flag.**

---

## Features

- Story-driven Mission Impossible themed challenge
- Multi-stage exploitation path
- Vulnerable Apache + PHP web application
- Privilege escalation via SUID misconfiguration
- Realistic penetration testing scenario
